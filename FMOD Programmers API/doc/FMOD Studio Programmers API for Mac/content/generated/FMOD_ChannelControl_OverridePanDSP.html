Firelight Technologies FMOD Studio API

ChannelControl::overridePanDSP

Replaces the built in panner unit FMOD uses per ChannelGroup and Channel, with a user selected panner. Can also be used to revert the panner back to the built in panner.

C++ Syntax

FMOD_RESULT ChannelControl::overridePanDSP(
  DSP *panner
);

Parameters

panner

Pointer to a FMOD_DSP_TYPE_PANNER dsp unit. Specify 0 or NULL to make FMOD use the built in panner.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

A panner is a unit that a channel uses to control the pan from setMixMatrix, setPan, and from parameters like 3D position.
A panner unit must support the enumerations specified by the FMOD_DSP_PAN enumeration list.
Use ChannelControl::getDSP with FMOD_CHANNELCONTROL_DSP_INDEX enum FMOD_CHANNELCONTROL_DSP_PANNER to get access to the built in panner.

Firelight Technologies FMOD Studio API

System::getChannelsReal

Retrieves the total number of playing channels that are non-virtual.

C++ Syntax

FMOD_RESULT System::getChannelsReal(
  int *realchannels
);

C Syntax

FMOD_RESULT FMOD_System_GetChannelsReal(
  FMOD_SYSTEM *system,
  int *realchannels
);

C# Syntax

RESULT System.getChannelsReal(
  out int channels
);

Parameters

realchannels

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
